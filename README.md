# CS355-02.Mario
Group Project by implementing a mario game in C. It is not a 1-1 adaption of Super Mario Bros. but it includes similar functionality.

– Mario moves at a constant speed that is determined at the beginning of the game.

– Mario has the ability to jump up to avoid incoming obstacles.

– If Mario consumes a Star that randomly appears in the game environment, Mario gains
the ability to run through the obstacles.

– After a reasonable time of game play, a castle exists in the game environment.

• Game Play: The game ends with a win if Mario reaches the castle. The game ends in lost if
Mario hits an obstacle before reaching the castle
